# keras-webapp 
## Started
- clone this project
-  install  requirements.txt
- run this project
- View http://localhost:5000
## Local installation
###### clone
- git clone https://github.com/Axzc/keras-webapp.git
###### install requirements
- pip install -r requirements.txt
###### Run with Python
Python 2.7 or 3.6 
  $ python app.py
