# keras-webapp 

## 使用
- clone 这个项目
 git clone https://github.com/Axzc/keras-webapp.git
-  安装  requirements.txt
 pip install -r requirements.txt
- 运行
Python 2.7 or 3.6 
$ python app.py
- 在 http://localhost:5000查看


## Docker 安装
###### 从 Dockerhub上拉取
$ docker pull axzc/flask-keras-web-app1
 &nbsp;
$  docker run -d -p 5000:5000 axzc/flask-keras-web-app1
